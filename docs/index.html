Oxygen is a multi-tenant video transcoding service built with Laravel 13 + Inertia v3 + React 19 on the web layer, a standalone Go worker for FFmpeg-based transcoding, and a separate Go live service for live-stream runtime concerns. VOD transcoding coordinates through shared Postgres, Redis, and S3. Live streaming uses Laravel as the control plane and the Go live service for publish auth proxying, session callbacks, HLS serving, and viewer rollups.

Architecture Overview

Oxygen has two runtime paths: VOD transcoding and live streaming. Laravel is the control plane for both.

Live streaming uses Laravel for stream records, encrypted keys, admin UI, and viewer metrics. The standalone Go live service handles RTMP ingest, validates publishers against Laravel, remuxes the stream into HLS, serves playback, and reports session and viewer state back to Laravel.
